# Activation Function

A mathematical function that introduces non-linearity into a neural network. Common examples:

- **Sigmoid** (logistic): \\(\sigma(x) = 1/(1+e^{-x})\\)
- **Tanh**: \\(\tanh(x) = (e^x - e^{-x})/(e^x + e^{-x})\\)
- **ReLU**: \\(f(x) = \max(0, x)\\)
- **Softmax**: \\(f(x_i) = e^{x_i}/\sum_j e^{x_j}\\)

Activation functions allow networks to learn non-linear mappings.
