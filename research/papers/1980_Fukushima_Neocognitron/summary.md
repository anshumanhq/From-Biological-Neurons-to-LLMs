# Summary: Neocognitron

**Author:** Kunihiko Fukushima  
**Year:** 1980  
**DOI:** 10.1007/BF00344251

Fukushima proposed the Neocognitron, a hierarchical network inspired by the visual cortex, using local receptive fields, weight sharing, and pooling to achieve shift invariance. The network learns feature detectors via unsupervised competitive learning. This was the architectural blueprint for all future convolutional neural networks, though it lacked backpropagation.
