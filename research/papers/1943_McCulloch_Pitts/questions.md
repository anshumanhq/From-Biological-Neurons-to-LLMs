# Open Questions: McCulloch & Pitts (1943)

1. The MP neuron has no learning mechanism. How would you extend it to include plasticity?
2. The model assumes binary, deterministic firing. How would stochastic firing change the computational power?
3. Can the MP neuron model be considered a valid abstraction of biological neurons today?
4. If the MP neuron is equivalent to a finite automaton, what is the simplest network that can simulate a Turing machine?
