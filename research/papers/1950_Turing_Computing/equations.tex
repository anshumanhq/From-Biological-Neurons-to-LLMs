% Turing (1950) – Formal Model of the Digital Computer (Universal Machine)
% 
% This is the transition function of a finite automaton / Universal Turing Machine.
% It underpins all of Turing's arguments in the 1950 paper.

\begin{equation}
\boxed{\delta: (Q \times \Sigma) \rightarrow (Q \times \Gamma \times \{L, R\})}
\end{equation}

Where:
\begin{itemize}
    \item $Q$: Finite set of internal states
    \item $\Sigma$: Input alphabet
    \item $\Gamma$: Tape alphabet
    \item $L$: Move tape head left
    \item $R$: Move tape head right
\end{itemize}

This is the mathematical abstraction of the "Discrete-State Machine" that Turing argues can simulate human thinking in principle.
