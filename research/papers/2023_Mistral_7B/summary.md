# Summary: Mistral 7B

**Authors:** Albert Q. Jiang et al.  
**Year:** 2023

Mistral 7B introduced a 7B-parameter efficient language model with Grouped-Query Attention (GQA) and Sliding Window Attention (SWA). GQA reduces inference memory requirements, while SWA enables efficient long-sequence processing. The model outperforms Llama 2 13B on many benchmarks, demonstrating that efficiency can be achieved without sacrificing performance. Released under an open-weight licence, Mistral 7B significantly influenced the open-weight LLM ecosystem.
