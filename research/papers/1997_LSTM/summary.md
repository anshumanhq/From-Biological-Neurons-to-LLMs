# Summary: Long Short-Term Memory

**Authors:** Sepp Hochreiter, Jürgen Schmidhuber  
**Year:** 1997

LSTM introduces a gated memory cell that allows recurrent networks to learn long-range dependencies. The Constant Error Carousel preserves gradient flow over time, and the input, output, and forget gates control access to the memory. This architecture became the dominant approach for sequence modelling for nearly two decades before Transformers.
