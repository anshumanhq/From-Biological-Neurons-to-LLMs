# Summary: Long Short-Term Memory (1997)

**Authors:** Sepp Hochreiter, Jürgen Schmidhuber  
**Year:** 1997

The original 1997 LSTM introduced a memory cell with a Constant Error Carousel (CEC) and input/output gating, enabling more reliable learning of long-term dependencies in recurrent networks. This version did not have a forget gate – adaptive resetting was added later (1999/2000). The architecture became the basis for sequence modelling for nearly two decades.
