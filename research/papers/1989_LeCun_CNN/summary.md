# Summary: Backpropagation Applied to Handwritten Zip Code Recognition

**Authors:** Yann LeCun, B. Boser, J. S. Denker, D. Henderson, R. E. Howard, W. Hubbard, L. D. Jackel  
**Year:** 1989  
**DOI:** 10.1162/neco.1989.1.4.541

LeCun applied backpropagation to a convolutional architecture inspired by Fukushima's Neocognitron. The network achieved state-of-the-art accuracy on the USPS handwritten digit recognition task, demonstrating that CNNs were practical and trainable. This paper established the blueprint for all modern computer vision systems.
