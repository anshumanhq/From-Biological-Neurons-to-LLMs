# Summary: Finding Structure in Time

**Author:** Jeffrey L. Elman  
**Year:** 1991

Elman introduced the **Simple RNN** with hidden‑state recurrence, replacing the output feedback of the Jordan Network. This innovation enabled richer temporal representations and became the standard for sequence learning before LSTM. The network was demonstrated on language acquisition tasks, showing that it could learn grammatical structure without explicit rules. It laid the groundwork for modern sequence models.
